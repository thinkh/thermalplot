ThermalPlot

Visualizing Multi-Attribute Time-Series Data Using a Thermal Metaphor

About

ThermalPlot summarizes complex combinations of multi-attribute time-series (e.g., interest rate or gross domestic product (GDP)) in a single degree-of-interest (DoI) value, describing the user-defined interestingness for each item. This is especially useful to provide an overview over a large collection of items, such as the economic development of countries.

The interestingness (DoI) is mapped to the x-axis in the plot, whereas the relative change in the DoI value (ΔDoI) within a user-speciﬁed time window (e.g., 3 month or 1 year) is mapped to the y-axis. Consequently each quadrant of the plot has an inherent meaning:

  • Quadrant I (top-right): items with a constant positive development
  • Quadrant II (top-left): items that had a negative phase in the beginning of the time window, but show a recent positive development
  • Quadrant III (bottom-left): items with a constant negative development
  • Quadrant IV (bottom-right): items that had a positive phase in the beginning of the time window, but show a recent negative development

Based on the interestingness (and therefore position) we adapt the items’ level of detail, to help the user to identify important items and reduce the visual clutter.

For the practical use, a couple of support visualizations are required. As illustrated in the Figure, the overall setup consists of four linked components: (a) the ThermalPlot as the core of our system, (b) the timeline showing the full time range for which data is available lets the user set an index point which the DoI value is compared to and a time window that speciﬁes the data which serves as an input to ThermalPlot, (c) the DoI editor as an interface for composing the computation function by means of combining and weighting multiple attributes, and (d) the detail view presenting the development of attributes over time for the selected items.
